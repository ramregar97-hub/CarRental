# Project Name: MERN Stack Car Dealership

## Description

This project is a full-stack web application for a car dealership, built with the MERN stack (MongoDB, Express, React, Node.js). It allows users to browse, add, edit, and delete car listings. The application features user authentication and a responsive user interface.

## Features

-   **User Authentication:** Secure user registration and login functionality.
-   **Browse Cars:** View a list of all available cars.
-   **Car Details:** See more information about a specific car.
-   **Add Cars:** Authenticated users can add new car listings.
-   **Edit Cars:** Authenticated users can edit their own car listings.
-   **Delete Cars:** Authenticated users can delete their own car listings.
-   **My Cars:** Users can view a list of the cars they have listed.
-   **Theme Toggle:** Switch between light and dark mode for a better user experience.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/your-repository.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd your-repository
    ```
3.  **Install backend dependencies:**
    ```bash
    cd backend
    npm install
    ```
4.  **Install frontend dependencies:**
    ```bash
    cd ../frontend
    npm install
    ```

## Available Scripts

### Frontend

In the `frontend` directory, you can run the following commands:

-   `npm run dev`: Starts the development server.
-   `npm run build`: Builds the app for production.
-   `npm run lint`: Lints the code.
-   `npm run preview`: Previews the production build.

### Backend

In the `backend` directory, you can run:

-   `npm start`: Starts the backend server.
